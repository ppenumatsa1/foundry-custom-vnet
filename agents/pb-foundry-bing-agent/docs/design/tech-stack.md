# Tech Stack

- Python 3.11+
- Azure AI Projects SDK (`azure-ai-projects`)
- Azure Identity (`DefaultAzureCredential` via `azure-identity`)
- Foundry hosted agent versions (`PromptAgentDefinition`)
- Foundry web grounding tool (`WebSearchPreviewTool`)
- Pydantic v2 + `pydantic-settings` for configuration
- `python-dotenv` for local `.env` loading behavior
- `pytest` for unit testing
- Makefile-based local developer workflow
